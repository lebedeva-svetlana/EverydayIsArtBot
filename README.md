# EverydayIsArt

Telegram bot that sends out random museum exhibits. All descriptions are in russian.

## Usage

Use /get command.

![Image](https://github.com/lebedeva-svetlana/EverydayIsArt/assets/91262515/cce44f43-c7b4-40db-ab19-8740c5840bf5)

## Used museums

- The State Tretyakov Gallery

Work in progress.
